# 🛡️ Cyber Risk Assessment & Threat Intelligence Platform

## 📌 Overview
A Python-based cybersecurity platform that:
- Performs vulnerability scanning
- Integrates threat intelligence APIs
- Calculates cyber risk scores
- Generates reports and dashboards

---

## 🚀 Features
- 🔍 Nmap-based vulnerability scanning
- 🌐 Shodan & VirusTotal integration
- 📊 Risk scoring engine (CVSS-inspired)
- 📄 PDF report generation
- 📈 Interactive Streamlit dashboard

---

## 🛠️ Tech Stack
- Python
- Streamlit
- Nmap
- Shodan API
- VirusTotal API
